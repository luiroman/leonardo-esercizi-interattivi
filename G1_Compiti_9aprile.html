2️⃣ Installazione Wireshark & Cattura Traffico

🔧 Cosa fare:

Installa Wireshark, cattura 5 minuti di traffico di rete, salva il file .pcap

PRIORITÀ ALTA

📥 Installazione

Windows / Mac

  1. Visita https://www.wireshark.org
  2. Scarica la versione per il tuo SO
  3. Installa (accetta le impostazioni di default)
  4. Apri l'applicazione

Linux (Ubuntu/Debian)

sudo apt update sudo apt install wireshark wireshark &

📡 Cattura Traffico (Primi Passi)

  1. Apri Wireshark
  2. Scegli scheda di rete: cerchi "Wi-Fi" oppure "Ethernet" (quella con pallino verde)
  3. Clicca il pulsante blu "Start" (icona di shark in alto a sinistra)
  4. Naviga normalmente (apri browser, YouTube, email, etc.) per ~5 minuti
  5. Clicca "Stop" (pulsante rosso)
  6. File → Export As → nomina il file:
Cognome_Wireshark_Cattura.pcap
✅ Non abbiate paura! Vedrete migliaia di pacchetti scorrere. È normale! Vedrete DNS, TCP, HTTP/HTTPS, ARP. Non dovete capire tutto ora—domani analizziamo insieme.

📁 Deliverable:

Portate il file .pcap domani (10 aprile) a Napoli, fermata Toledo. O inviate via email al docente.

3️⃣ Letture Opzionali (Se Avete Tempo)

📖 Routing Statico vs Dinamico (Introduzione)

Domanda: Come un router sa dove mandare un pacchetto?

Risposta breve:

  • Routing statico: L'admin configura manualmente le rotte
  • Routing dinamico: Router comunicano fra loro (OSPF, BGP). Automatico.

Per domani: Capite il concetto. Non servono dettagli tecnici—vedremo i comandi live in aula.

⏱️ Tempo: ~20 minuti

💾 VirtualBox + Ubuntu Server (Molto Opzionale)

Se volete lab avanzati (non obbligatorio):

  1. Scarica VirtualBox da https://www.virtualbox.org
  2. Scarica Ubuntu Server 24.04 LTS da https://ubuntu.com/download/server
  3. Crea una VM (RAM: 2-4GB, Disco: 20GB)
  4. Non avviate se non sapete cosa fare—aspettate lezioni su lab avanzati
